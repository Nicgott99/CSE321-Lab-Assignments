## How to Compile and Run

Each assignment contains C source files that can be compiled using gcc:

```bash
# Navigate to the specific assignment directory
cd CSE321_Lab_Assignment_X/

# Compile the C file
gcc -o task1_exec task1.c

# Run the executable
./task1_exec
```
